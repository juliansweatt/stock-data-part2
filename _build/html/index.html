get_tickers_from_url(url, maximum, currentTickers)¶

Gets stock tickers from a specified NASDAQ url.

Details

Function retrieves a maximum amount of tickers from a specified URL. A NASDAQ URL is expected. The maximum quantity takes in to account the current amount of tickers already found outside the function. The function will return whenever the maximum is reached or the page tickers are exhausted. The intent is to feed this function each successive page URL.

Parameters

  • url (str) – URL to retrieve stock data from.

  • maximum (0 <= int <= 110) – The maximum number of tickers to retrieve.

  • currentTickers (int) – The current amount of tickers already retrieved.

Returns

Retrieved tickers.

Return type

set of str
